Требования

Системные требования

OmegaT будет работать на любой системе, в которой была или может быть установлена среда JRE (Java Runtime Environment). В настоящий момент среда JRE поставляется вместе с OmegaT, и её необязательно загружать отдельно. OmegaT успешно устанавливается в операционную систему Windows (все версии, начиная с 98), Macintosh OS X и GNU/Linux.

Файлы в формате Microsoft Office XML можно переводить непосредственно в OmegaT. Файлы в более старых форматах MS Word, Excel и Powerpoint можно переводить, предварительно преобразовав их в актуальные форматы Microsoft Office или в формат Office Open XML, что можно сделать в OpenOffice.org или LibreOffice (тоже бесплатный). Загрузить OpenOffice.org или LibreOffice можно по ссылке в разделе «Сторонние программы».

Требования к пользователям

OmegaT несложно установить или использовать, но вы должны быть готовы к чтению документации. Если вы питаете отвращение к чтению документации, а под изучением новых программ подразумевается нажимание на все кнопки до тех пор, пока не появится нечто, похожее на желаемое, OmegaT, вероятно, не подойдёт для вас.

Устаревшие версии

You can still find older, superseded versions of OmegaT.

Устаревшие версии — это версии, которые были замещены более новыми версиями. Для большинства пользователей они не представляют никакого интереса. Однако некоторые пользователи могут столкнуться с ограничениями (наложенными операционной системой или версией Java), что может ограничить их выбор только устаревшими версиями.

С/без JRE

In order to run, OmegaT requires the Java Runtime Environment (JRE) to be installed on your computer.

Если вы уверены, что подходящая версия JRE уже установлена на вашем компьютере, вы можете загрузить OmegaT без JRE. Загрузка произойдёт быстрее, а для установки потребуется меньше места на вашем диске. В данном случае не нужно выбирать версию по умолчанию, а выбрать другую версию (например стандартную или последнюю), а затем выбрать версию "без JRE".

Если вы установите версию OmegaT с JRE, даже если на вашем компьютере уже установлена JRE, то JRE в составе OmegaT будет установлена отдельно и не будет с ней конфликтовать. Таким образом, установка OmegaT с JRE всегда безопасна. Если вы не уверены, есть ли на вашем компьютере подходящая версия JRE, выбирайте версию с JRE (или версию по умолчанию).

Mac OS X users: the JRE is installed on Mac OS X by default; there is therefore no version for Mac OS X with the JRE.

Linux users: most Linux distributions are supplied with a Java Runtime Environment, but some open-source implementations do not work properly with OmegaT (although this may not be immediately apparent). Если вы относительно недавно используете Linux, мы советуем вам выбирать версию OmegaT с JRE. Более опытные пользователи могут захотеть установить JRE от Oracle (ранее Sun) на системном уровне.

Signed/Unsigned

Mac OS X has a feature called Gatekeeper that increases security by making it difficult to run unsigned programs. OmegaT for Mac OS X is available in both signed and unsigned versions; they are functionally identical, but the signed version is preferred as it is recognized by the system as being published by a known developer.

There may be a delay in publishing signed versions. Users in a hurry can use the unsigned version instead; if you have difficulty launching the app, see how to open an app from an unidentified developer.

Версия Web Start предлагает альтернативный способ установки OmegaT на ваш компьютер.

Для этой версии все последующие обновления будут устанавливаться автоматически.

В зависимости от вашей операционной системы в процессе установки вы можете увидеть несколько сообщений, касающихся безопасности.
Сертификат подписан PnS Concept Sarl, компанией, зарегистрированной во Франции.

Разрешения, которые вы даёте этой версии (которые могут быть описаны как "неограниченный доступ к компьютеру"), ничем не отличаются от разрешений для локальной версии, т. е. доступ к жёсткому диску вашего компьютера.

Чтобы установить версию Web Start, на вашем компьютере уже должна стоять Java версии 1.3 или более поздняя.

Информация о конфиденциальности: OmegaT Java Web Start не отправляет никаких ваших данные на сервера OmegaT. Приложение работает на вашем компьютере. Ваши документы и памяти переводов остаются на вашем компьютере, а проект в OmegaT не имеет доступа к вашим личным данным.

Версия из /trunk (альфа-версия) отражает наиболее актуальное состояние процесса разработки программы. Она рекомендуется только для серьёзных разработчиков и тестировщиков. Также её можно получить, выполнив команду subversion:

svn checkout svn://svn.code.sf.net/p/omegat/svn/trunk omegat-svn

Переодические сборки доступны на CloudBees.
